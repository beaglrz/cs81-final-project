# Hello Kitty Memory Match Game

# Project Summary
This project is a Hello Kitty Memory Match Game made with React and Vite. Players flip cards to find matching pairs of Sanrio characters. The game counts your moves and shows a message when you match all the pairs.

# Key Features and Functionality
- Interactive memory matching game
- Flip cards to reveal Sanrio characters
- Match detection with automatic flip-back for mismatches
- Keep track of your moves with a counter that updates every turn
- “New Game” button to reset and reshuffle the deck
- Win message displayed when all pairs are matched
- Responsive grid layout with square cards that works on any device
- Each card back features a bow image

# Technologies Used
- React
- Vite
- JavaScript
- HTML/JSX
- CSS

# Instructions
To get started, you can clone the repository to run the project on your computer, or you can play the game right away using the live GitHub Pages link.

How to Run Locally
1. Open your preferred code editor (e.g., Visual Studio Code)
2. Clone the repository using a terminal (git clone https://github.com/beaglrz/cs81-final-project.git)
3. Navigate into folder (cd cs81-final-project)
4. Install dependencies (npm install)
5. Start DEV server (npm run dev)
6. Open localhost link (http://localhost:5173)
